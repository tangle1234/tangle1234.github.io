小米手机刷机

发表于2017-09-05|玩机
|字数总计:417|阅读时长:1分钟|阅读量:|评论数:

环境准备

  1. 已解锁小米手机一部；
    • 解锁点击申请解锁
    • 通过申请后下载解锁工具进行解锁
  2. 电脑一台；
  3. 驱动已安装好；
  4. 刷机包
    • 线刷包下载地址官网下载；
    • 卡刷包下载地址官网下载；
    • 第三方包可以从智友，移动叔叔等论坛下载；
  5. 资料备份好
    • 可以通过自带备份功能备份；
    • 其他方式备份；
  6. 刷机工具miflash
    • 下载地址同线刷包下载地址

开始刷机

卡刷

  1. 将卡刷包放进内存；
  2. 进入recovery模式；
    • 长按电源键+音量加键
  3. 双请，比较大的版本更新需要三清；
  4. 根据提示选择刷机包刷机；
  5. 重启，需等待较长时间。

线刷

  1. 打开线刷工具miflash；
  2. 选择好刷机包（需要解压）；
  3. 手机进入fastboot模式
    • 长按电源键+音量减键
  4. 连接手机，点刷新，选择设备，清除哪些东西，选择好后开始刷机；
  5. 重启。

三方包卡刷

  1. 刷入三方recovery；
    • Twrp官网下载对应机型的recovery
    • 下载并解压三方recovery刷入工具点击下载，提取码grlb
    • 用文本编辑器打开recovery-twrp一键刷入工具，将set rec_img=recovery-TWRP-3.3.1-0529-REDMI_K20PRO-CN-wzsx150.img，“=”后面的内容改为从官网下载的recovery的名字，保存
    • 手机进入fastboot模式连接，再运行recovery-twrp一键刷入工具选择刷入
  2. 刷入magisk框架root，如果不root，重启手机，刷入的recovery就会没了
  3. 其他步骤跟官方卡刷一样，将刷机包放到手机存储里，进入recovery模式选择刷入。
