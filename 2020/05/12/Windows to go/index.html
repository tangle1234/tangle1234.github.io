Windows to go

发表于2020-05-12|玩机
|字数总计:228|阅读时长:1分钟|阅读量:|评论数:

准备

  1. U盘1个，最好大于16G，速度要快一点的，不然运行起来很卡，现在USB3.0的U盘很便宜了
  2. Windows镜像文件，可以直接去MSDN，我告诉你下载
  3. WTG辅助工具，点击下载，提取码h3aq
  4. Win系统电脑一台

开始制作

  1. 解压镜像文件

  2. 打开WTG辅助工具，选择镜像解压目录下的/source/install.vim

  3. 选择安装win版本

  4. 右边可以选择UEFI+GPT或UEFI+MBR，分区可选择分1-3个区

  5. 点击部署，等在完成就制作好了

启动

进电脑UEFI设置页面设置U盘启动，启动后根据提示设置好各种选项

驱动安装

  1. 在磁盘管理选择电脑的磁盘联机
  2. 在设备管理看哪些是黄色的，选择更新驱动，目录选择原电脑系统盘的Windows/system32
  3. 等待驱动安装完成
